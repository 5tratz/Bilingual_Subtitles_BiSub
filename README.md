# AI Text Analyzer (AiTA)

A comprehensive, professional-grade tool designed for educators to detect AI-generated content in student submissions with 99%+ accuracy using multiple verification methods.

<b>Features</b>:

<b>Core Analysis</b>
- Advanced AI Detection - Multiple verification methods for maximum accuracy.
- 12+ Analysis Metrics - Comprehensive text analysis including perplexity, burstiness, and readability.
- Multi-Format Support - TXT, PDF, DOCX, RTF, ODT, and image files (JPG, PNG, BMP).
- Batch Processing - Analyze multiple files simultaneously.
- Visual Analytics - Real-time charts and trend visualization.

<b>User Interface</b>
- Dark/Light Theme - Eye-friendly interface for any environment.
- UI Size Scaling - Small, Medium, or Large interface.
- Optional Tooltips - Guided assistance (disabled by default).
- Real-time Results - Instant feedback with visual indicators.
- Grade System - A+ to F grading based on AI probability.

<b>Data Management</b>
- Analysis History - Complete history with search functionality.
- Storage Management - Keep 25, 50, or unlimited records.
- Export Options - Export history to JSON or CSV.
- Automated Backup - Create and restore backups.
- Organized Storage - Automatic report organization.

<b>Reporting</b>
- HTML Reports - Professional, printable reports with teacher signature.
- PDF Export - Official documentation format (requires reportlab).
- Teacher Sign-off - Add your name to reports.
- Confidence Scoring - Each result includes confidence level and validation score.

<b>Security</b>
- Hardware-Bound License - License tied to machine hardware.
- Secure Activation - 20-character license key system.
- Data Privacy - All data stored locally in user directory.
